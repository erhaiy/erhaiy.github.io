HTML的表单标签

表单：用于采集用户输入的数据的，用于和服务器交互
from：用于定义表单的，可以定义一个范围，范围代表采集用户数据的范围
属性
action：指定提交数据的URL
method： 指定提交方式
分为7种类，常用两种：get和post

1.get提交地址栏会携带提交的数据，post不会携带
2.get请求安全级数较低，post较高
3.get请求数据大小有限制，post没有

演示；

这方式为post，点击注册后会将账号和密码显示在地址栏中，而get则不会。注意：表单数据提交的时候，需指定name属性。

代码： 

<form method="get">
账号：<input type="text" name="name"/><br/>
密码：<input type="password" name ="pwd"/><br/>
<input type="submit" value="注册"/>
<input type="reset" value="重置"/>
</form>

另外再介绍几种HTML常用的标签

重置按钮:<input type="reset" value="重置"/>

普通按钮:<input type="button" value="按钮名称"/>

加粗: <b>加粗的内容</b>

删除线: <s>内容</s>

下划线: <u>内容</u>

段落标签：<p>内容</p>

斜体：<i>内容</i>

div:<div>自动换行</div>

在一行内显示<span>内容</span>

CSS

CSS的简介

CSS：层叠样式表

层叠：一叠一叠的
样式表：很多的属性和属性值
让页面变得更好看，CSS将网页内容和显示样式进行分离，提高了显示功能

CSS和HTML的结合方式

内联样式
在每个HTML标签上都有一个属性style，将CSS与HTML结合起来 

<div style="color=red;">hello css</div>

内部样式
在head标签内，定义style标签，style标签的标签体的内容为CSS代码 

<style>
    div{
        color:blue;
    }
</style>
<div>hello css</div>

外部样式
1.定义css资源文件
2.在head标签内，定义link标签，引入外部的资源文件
如：a.css文件： 

div{
    color:green;
}

HTML文件中； 

<div>hello css</div>

注意：
①1，2，3种方式，css作用范围越来越大
②第一种方式不常用，后期常用2，3
③第三种格式可以写为：
<style>
@inport "css/a.css";
</style>

CSS的选择器

筛选具有相似特征的元素
基础选择器
① id选择器 

——<div id ="haha">AAA</div> 
——#haha {
    background-color:black;
}

② class选择器 

——<div class="hehe">BBB</div>
——.hehe{
    background-color:green;
}

③ 标签选择器 

——<div>CCC</div>  
——div{
    background-color:gray;
}

扩展选择器
①关联选择器 

——<div><p>DDD<p><div> 
——div p{
    background:color:green;
}

②组合选择器 

——<div>111</div>  
  <p>222</p>
——div,p{
    background-color:black;
}

③伪元素选择器
超链接状态：

 ——<a>链接文字</a>   
 ——
 原始状态:a.link{}
 悬浮状态:a.hover{}   
 点击状态:a.active{}   
 点击之后:a.visited{}

CSS的盒子模型

在进行布局前需要把数据封装为一块一块的区域内


1⃣️ 边框
border： 2px solid blue;
border:统一设置
上 border-top
下 border-bottom
左 border-left
右 border-right
2⃣️ 内边距
padding:20px;
使用padding分别统一设置，如border也可分别设置
3⃣️ 外边距
margin： 20px；
可以使用margin统一设置，如border也可分别设置

CSS的漂浮模型

float：属性值
left：文本流向对象的右边
right：文本流向对象的左边
