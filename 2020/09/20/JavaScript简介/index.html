JavaScript

简介

基于对象和事件驱动的语言，应用于客户端(游览器)
特点
①交互性：信息的动态交互
②安全性：JavaScript不能访问本地磁盘的文件
③跨平台性:支持JS的游览器都可以运行
Java和JavaScript的区别
①JS是基于对象的，Java是面向对象的
②Java是强类型语言，JS是弱类型语言

强类型语言:一个类型的变量一旦被定义了，如果不经过强制转换，那么它永远就是这个数据类型
弱类型语言：一个变量可以赋不同类型的值
var a = 1; var b = "字符";

JavaScript的组成
①ECMA Script
—ECMA:欧洲计算机协会
—由ECMA组织指定的js的语言，语句
②BOM
—Browser object model：游览器对象模型

③DOM
—Document object model：文档对象模型

JS与HTML的结合

第一种： 

<script type="text/javascript">JS代码</script>

第二种： 

<script type="text/javascript" src="路径"> 
不要在此写JS代码（因为不会运行）
</script>

JS的原始类型和声明变量

5种原始类型
① String
var str = "abc";
② number
var n =123;
③ boolean
var flag =true;
④ null
var a =null;
⑤ undifined(定义一个变量，没有赋值)
var aa;
查看变量的类型
typeof(变量名称);

JS的语句

JS种有if，switch，for ，while，do…while语句

JS的运算符

==和===的区别

1.都是做判断用的
2.==比较的是值，===比较的是值和类型
直接向页面输出变量和或H5代码
document.write();

JS的数组

第一种： 

var arr =[1,2,3];
var arr =[1,"2",true];

第二种:使用内置对象，Array对象 

var arr1 = new Array(5);
arr1[0]="1";

第三种： 

    var arr2 = new Array(3，4，5);
    定义一个数组，其元素为3，4，5

JS数组的属性
① length
② 长度可变
③ 可存放不同的数据类型

JS的函数

① 使用关键字function 

function 方法名(参数列表){
方法体;
返回值可有可无；
}

② 匿名函数 

var add = function(参数){
    方法体;
}

JS的String对象

① 创建String对象 

var str = "abc";

② 方法
与HTML相关的方法
document.write()为打印方法，为了理解这些方法，使用了打印效果 

//加粗
document.write(str.bold());
//设置字体的颜色   
document.write(str.fontcolor("red"));
//设置字体的大小   
document.write(str.fontsize(5));
//将字符串显示成超链接效果   
document.write(str.link());
//下标，上标   
document.write(str.sub());
document.write(str.sup());

与Java类似的方法
——concat；连接字符串 

    document.write(str1.concat(str2));

——charAt()；返回指定位置的字符串 

document.write(str.charAt(5));

——indexof()；返回字符串的位置 

document.write(str.indexof("a"));

——split();切分字符串成数组 

    var str ="a-b-c";
    var str1 =str.split("-");//a,b,c

——replace();替换字符串 

document.write(str.replace("a","q"));
//将q替换为a

——substr()和subString 

str.substr(5,3);//从第5位到向后的3位
str.subString(3,5);//从第3位到第5位

JS的Array对象

属性：length
方法
数组的连接： 

var arr1 = [1,2,3];
var arr2 = [4,5,6];
document.write(arr.conncat(arr2));

根据指定的字符分割数字 

document.write(arr1.join("-"));

向数组的末尾添加新元素，返回数组的长度 

 var a =arr1.push("a");

删除最后一个元素，返回删除的元素 

var b = arr1.pop();

反转元素 

document.write(arr1.reserve());

JS的Math对象

数学的运算
里面都是静态方法。使用Math.方法()
向上舍入：Math.ceib(x); 如10.5=11
向下舍入：Math.floor(x)
四舍五入：Math.round(x)
得到一个随机数：Math.random();
x的y次方：Math.pow(x,y)
