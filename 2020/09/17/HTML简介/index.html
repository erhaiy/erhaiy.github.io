Web概念概述

Java Web：使用Java语言开发基于互联网的项目

软件架构

1.C/S：Client/Server：客户端/服务器端
在用户本地有一个客户端程序，在远程有一个服务器端程序。如：QQ
优点：1.用户体验好
缺点：1.开发，安装，部署，维护麻烦
2.B/S：Brower/Server：游览器/服务器端
只需要一个游览器，用户通过不同的网址（URL），客户端访问不同的服务器端程序
优点：1.开发，安装，部署，维护简单
缺点：1.如果应用过大，用户的体验可能会收到影响
2.对硬件要求额高
B/S架构详解
资源分类：
1静态资源：使用静态网页开发技术发布的资源
特点：
1⃣️ 所有用户访问，得到的结果是一样的
如：文本，图片，音频，视频，HTML，CSS
2⃣️ 如果用户请求的是静态资源，那么服务器会直接将静态资源发给游览器。游览器中内置了静态资源的解析引擎可以展示静态资源
2:使用动态网页
使用动态网页及时发布的资源
特点：
1⃣️ 所有用户访问得到的结果可能不一样
如：Jsp/Servlet，php，asp
2⃣️ 如果用户请求的是动态资源，那么服务器会执行动态资源，转换为静态资源，再发送给游览器
静态资源
HTML：用于搭建基础网页，展示页面内容
CSS：用于美化页面，布局页面
JavaScript：控制页面的元素，让页面有一些动态的效果

HTML基础

1.概念：

最基础的网页开发语言
Hyper Text Markup Language
A；超文本：超出文本的范畴，使用html可轻松实现这样的操作
B：标记：html所有的操作都是通过标记实现的，标记就是标签
C：网页语言：

2.html的规范

1⃣️ 一个html文件开始标签和结束的标签 

<html></html>

2⃣️ html包含两部分内容： 

<head>设置相关信息<head>    
<body>显示在页面上的内容都写在body里面</body>

3⃣️ 标签要成对存在，又开始也有结束
4⃣️ html的代码不区分大小写
5⃣️ 有此标签，没有结束标签 

如： <br/>

3.html的操作思想

使用标签将其包装起来，通过修改标签的属性值实现标签内数据样式的变化
一个标签相当于一个容器，想要修改容器内数据的样式，只需要改容器的属性值

HTML常用的标签

1.文字标签：修改文字的样式

    <font></font> 
    属性：  
    size：文字的大小  取值：1～7   
    color：文字颜色
        1.英文单词：blue，black   
        2.采用十六进制数表示：#ffffff

2.注释：

  <!-- 注释 -->

3.标题标签，水平线标签和特殊字符

标题标签：<h1></h1>
        <h2></h2>
        ......
        <h6></h6>
水平线标签：<hr/>   
        属性：size  color
           eg：<hr size="5" color="blue"/>   
特殊字符： < : &lt     > : &gt  
        空格: &nbsp   & : &amp

4.列表标签

1⃣️dl标签：

    <dl>
        <dt>传智播客</dt>
        <dd>财务部</dd>
        <dd>人事部</dd>  
    </dl>

2⃣️.ol标签：

<ol>
    <li>财务部</li>
    <li>学工部</li>
    <li>人事部</li>
</ol>

3⃣️，有特殊符号 

<ul type = disc>
    <li>财务部</li>
    <li>学工部</li>
</ul>
*type: 空心圆：circle   实心圆：disc（默认） 
       空心方块：square

5.图像标签

<img src="图片的路径" 其他/>  
其他：wide=” “
     height=” “   
     alt=”图片上的文字 “

6.超链接标签

1⃣️链接资源 

<a href="路径" target=“_blank” 文字>  </a>

2⃣️定位资源
回到顶部：

<a href="#top">链接中的文字</a>

3⃣️原样标签

<pre> </pre>

7.表格标签

<table border="2" bordercolor="" width="200" height="200" cellspacing="0">
<tr> <td>一号</td> <td>二号</td> </tr>
<tr> <td>内容</td> <td>内容</td> </tr>
</table>

参数
-table 表格线
-tablecolor 表格线颜色
-cellspacing 单元格之间的距离
-width 表格的宽度
-hight 表格的高度

居中加粗

<th> </th>

表格的表头

<caption>内容</caption>

合并单元格

跨列： <td rowspan="3">
跨行： <td clospan="2">

8.表单

普通输入项：

<input type="text" name="name"/>

密码输入项：

<input type="password" name="pwd"/>

单选输入项：

<input type="radio" check="check"/>
check为默认选中

复选输入项：

<input type="checkbox"/>   
在里面需要输入相同的name属性值

文件输入项：

<input type="file">

下拉输入项：

<select name="birth">
<option value="1991" select="select">1991</option>
<option value="1992">1992</option>
<option value="1993">1993</option>
</select>

select为默认选择项

文本域：

<textarea cols="100" rows="10">内容</textarea>
